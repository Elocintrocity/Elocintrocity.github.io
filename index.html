My Work

Novascape

Novascape is a 2D Unity game developed during CS196 as a semester-long project. The most prominent feature the game has is networking functionality and a procedurally generated terrain, assuring no playthrough is the same as before. This game was made by 2 other students and myself. I was in charge of the physics, networking troubles, and controls. See my GitHub page for the code evaluated at the end of the Fall 2015 semester.
